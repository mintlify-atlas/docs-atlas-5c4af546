## Get flights between two airports

<CodeGroup>
```bash cURL
curl -X GET "https://api.aero.com/v1/flights?from=LAX&to=JFK&date=2024-03-20" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.aero.com/v1/flights?from=LAX&to=JFK&date=2024-03-20', {
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.aero.com/v1/flights',
    params={'from': 'LAX', 'to': 'JFK', 'date': '2024-03-20'},
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)
data = response.json()
```
</CodeGroup>

<ParamField path="GET" type="/v1/flights">
  Get all flights between two airports for a specific date
</ParamField>

### Query parameters

<ParamField query="from" type="string" required>
  Departure airport IATA code
  
  **Example:** `LAX`
</ParamField>

<ParamField query="to" type="string" required>
  Arrival airport IATA code
  
  **Example:** `JFK`
</ParamField>

<ParamField query="date" type="string" required>
  Flight date in ISO format
  
  **Example:** `2024-03-20`
</ParamField>

## Get airline from flight number

<CodeGroup>
```bash cURL
curl -X POST "https://api.aero.com/v1/flights/airline" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"search": "6E9920"}'
```

```javascript JavaScript
const response = await fetch('https://api.aero.com/v1/flights/airline', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ search: '6E9920' })
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.aero.com/v1/flights/airline',
    json={'search': '6E9920'},
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)
data = response.json()
```
</CodeGroup>

<ParamField path="POST" type="/v1/flights/airline">
  Get airline with search query. This doesn't check if the number is correct.
</ParamField>

### Request body

<ParamField body="search" type="string" required>
  The search query, like flight number with airline code
  
  **Example:** `6E9920`
</ParamField>

### Response

<ResponseField name="id" type="string">
  Airline ID
</ResponseField>

<ResponseField name="name" type="string">
  Airline name
</ResponseField>

<ResponseField name="icao" type="string">
  ICAO code
</ResponseField>

<ResponseField name="iata" type="string">
  IATA code
</ResponseField>

<ResponseField name="image" type="string">
  Airline logo image URL
</ResponseField>

```javascript JavaScript
    'Authorization': 'Bearer YOUR_JWT_TOKEN'

});
```


</CodeGroup>

<ParamField path="GET" type="/v1/flights/tracked">
