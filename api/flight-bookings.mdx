## Create a flight booking

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:3000/v1/flight/booking" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "flightId": "flight_abc123",
    "bookingCode": "ABC123",
    "seatNumber": "12A",
    "seatType": "window",
    "seatingClass": "economy",
    "reason": "personal",
    "notes": "Window seat preferred"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:3000/v1/flight/booking', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    flightId: 'flight_abc123',
    bookingCode: 'ABC123',
    seatNumber: '12A',
    seatType: 'window',
    seatingClass: 'economy',
    reason: 'personal',
    notes: 'Window seat preferred'
  })
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'http://localhost:3000/v1/flight/booking',
    json={
        'flightId': 'flight_abc123',
        'bookingCode': 'ABC123',
        'seatNumber': '12A',
        'seatType': 'window',
        'seatingClass': 'economy',
        'reason': 'personal',
        'notes': 'Window seat preferred'
    },
    headers={'Authorization': 'Bearer YOUR_JWT_TOKEN'}
)
data = response.json()
```
</CodeGroup>

<ParamField path="POST" type="/v1/flight/booking">
  Create a new booking for a flight with booking details
</ParamField>

### Request body

<ParamField body="flightId" type="string" required>
  Flight ID to associate the booking with
  
  **Example:** `flight_abc123`
</ParamField>

<ParamField body="bookingCode" type="string">
  Booking code/confirmation code (max 50 characters)
  
  **Example:** `ABC123`
</ParamField>

<ParamField body="seatNumber" type="string">
  Seat number (max 10 characters)
  
  **Example:** `12A`
</ParamField>

<ParamField body="seatType" type="string">
  Type of seat position
  
  Options: `window`, `middle`, `aisle`, `jumpseat`, `captain`, `pilot`, `copilot`, `flight_engineer`, `flight_attendant`, `observer`, `other`
  
  **Example:** `window`
</ParamField>

<ParamField body="seatingClass" type="string">
  Class of service
  
  Options: `economy`, `premium_economy`, `business`, `first`, `private`, `other`
  
  **Example:** `economy`
</ParamField>

<ParamField body="reason" type="string" default="personal">
  Reason for the flight
  
  Options: `personal`, `business`, `crew`, `training`, `repositioning`, `other`
  
  **Example:** `personal`
</ParamField>

<ParamField body="notes" type="string">
  Additional notes about the booking (max 500 characters)
  
  **Example:** `Window seat preferred`
</ParamField>

  -d '{

    "seatType": "aisle",
    "notes": "Changed to aisle seat"

    seatType: 'aisle',
